WellBeing web app

React JS & Tailwind

Live version

This is a Wellbeing App that has been created based on inspiration from a paper notebook. It is built using React and Tailwind.

It includes features such as: list of priorities, today's focus, mood tracker with animations, daily hydration visualization where glasses fill up after clicking, possibility to generate affirmations using an API.
