Family ToDo List App

HTML, CSS & JS

Live version

This is a different version of a todo list, created for families. It allows for the creation of up to 5 members, where each member can choose their own color to differentiate their tasks. When creating a task, we select a member from the list, choose a day, and enter the task. The task is then added with a background assigned to that particular member.

It is created in HTML, CSS and Vanilla JS.
